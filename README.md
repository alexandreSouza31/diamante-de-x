### Funcionalidades✅ 
- Solicita ao usuário um número inteiro ímpar maior ou igual a 3.

- Valida a entrada do usuário, exibindo mensagens de erro em caso de:
    - Número par;
    - Número menor que 3.

- Mostra um Diamante de X, com:
    - A linha central contendo o número máximo de caracteres(número digitado).
    - O número total de linhas e colunas igual ao número ímpar fornecido.

- Opção de reiniciar após término

### Como rodar o código? 🤖

#### ❗❗Obs: Há a necessidade de ter o .NET SDK instalado em sua máquina previamente!

#### Passo a passo - Clone ou baixe o projeto  👞👞

1. Abra o terminal do seu editor de código;
2. Navegue até a pasta onde deseja instalar o projeto;
3. Clone o projeto 
ex:``` git clone git@github.com:alexandreSouza31/diamante-de-x.git```
 ou se preferir, baixe clicando no botão verde chamado "Code" no repositório desse projeto, e depois "Download zip.


#### Uso 💻
1. Inicie o App:
Certifique-se de estar na pasta do projeto, e navegue pelo terminal até o caminho do arquivo Program.cs
```
diamanteDeX\diamanteDeX.ConsoleApp>
```
2. Compile e execute o programa: ```dotnet run```

    ou, com o arquivo Program.cs aberto clique no botão verde(Current Document(Program.cs)) para iniciar

3. Siga as instruções no console para desenhar o Diamante!
